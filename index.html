streamsx.topology

The streamsx.topology API enables a developer to create IBM streaming applications entirely in Java, requiring no prior knowledge of SPL. Streamsx.topology employs a functional style of programming -- a developer may define a graph's flow and data manipulation simultaneously.

To set up the streamsx.topology api for use on a streams cluster, download the release from the releases page and untar it from your home directory. Ensure that your environment is properly configured: 


    $ cd ~/
    $ tar -xvzf streamsx-topology.tar.gz
    $ export STREAMS_INSTALL=/path/to/streams/install

Make sure that everything is set up correctly by compiling and running the sample HelloWorld application:


    $ javac -cp "~/streamsx-topology/com.ibm.streamsx.topology/lib/com.ibm.streamsx.topology.jar" simple/HelloWorld.java
    $ java -cp "~/streamsx-topology/com.ibm.streamsx.topology/lib/com.ibm.streamsx.topology.jar:${STREAMS_INSTALL}/lib/*:." simple.HelloWorld

Now that your environment is configured, you may begin using the API! Please refer to the getting started guide, FAQ page, and documentation for help.
